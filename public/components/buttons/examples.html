The default button styles
.button
Solid / Colors
The color classes are used to change the color of the button.
.button--success .button--warning .button--danger.button--info
Solid / Disabled
The disabled state is faded and disables cursor events.
.button--disabled
Solid / Loading
The loading state is faded, disables cursor events and animates the background.
.button--loading
Solid / Sizing
The sizing classes are used to change the size of the button.
.button--small .button--large
Solid / Block
The block class is used to make the button fill the width of the parent container.
.button--block
